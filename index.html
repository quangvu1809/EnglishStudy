AI Tạo Đề Thi

Nhập Google Gemini API Key (miễn phí) để AI tự động tạo đề thi mới. Câu hỏi AI tạo ra sẽ được lưu lại để mở rộng ngân hàng đề.

Hướng dẫn lấy API Key miễn phí
  1. Truy cập aistudio.google.com/apikey
  2. Đăng nhập bằng tài khoản Google
  3. Nhấn "Create API Key"
  4. Chọn project bất kỳ hoặc tạo mới → nhấn "Create"
  5. Copy key (dạng AIzaSy...) → dán vào ô bên dưới → nhấn Lưu

Gemini API hoàn toàn miễn phí. Hệ thống tự động thử nhiều model (flash-lite → 1.5-flash → 2.0-flash) nếu model nào bị giới hạn.
